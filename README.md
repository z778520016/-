# -
以山鬼、湘君等鬼神神话为核心，展示荆楚大地风采，体现楚人筚路蓝缕，在蛮荒之地建设，充满浪漫与奇特的旅程，奇幻而又缥缈。

# 主题
主要展现奇幻而又原始、充满荆楚（例如：暮霭沉沉楚天阔）的风景，与自然斗，与山精鬼怪同行，捉摸不透而又隐在雾中。

# 骨架（主要游戏内容）
1. 可以是采集草药、对抗疾病，草药的生成可以是随机的，能让人发挥创造，自己做草药，或者直接用现成的草药配方，由大模型验证配方效果，毕竟到现在这么多药，总之但肯定不脱离自然环境，有逻辑可循，也就意味着可以存档，但是没有绝对通关，可能运气不好，草药一直没有生成；
2. 帮助山鬼、湘君，互动
3. 实在不行，就画风景，把这些鬼怪形象、故事复制一遍，让人参与其中，与天斗。
4. 可以把每次通过的内容做成一个新的鬼神形象，让后续有参与、可以共同采草药之类。
5. 主要目的还是画风景吧，越逼真越好，尽量贴近现实（除了鬼神）。
